# TodoParul – React Todo App

This is a Todo List web application built using React and Vite.  
The application allows users to add and manage tasks easily.  
Firebase is used for configuration and backend connectivity.

## Features
- Add new tasks
- View the todo list
- Simple and clean user interface
- React component-based structure

## Technologies Used
- React
- Vite
- JavaScript
- CSS
- Firebase

## Project Structure
src/
  AddTodo.jsx
  Todolist.jsx
  App.jsx
  firebaseconfig.js

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/TODOPARUL.git
```

2. Install dependencies

```
npm install
```

3. Run the development server

```
npm run dev
```
